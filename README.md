## 🎯 Tujuan Pembelajaran

Tujuan mempelajari *data science* adalah untuk mengembangkan kemampuan dalam mengolah informasi menjadi landasan keputusan yang strategis. Di lingkungan pekerjaan saat ini, hampir semua sektor industri sangat bergantung pada analisis data untuk memecahkan masalah operasional dan mengoptimalkan proses bisnis mereka.
